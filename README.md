# Vocab Cards

A lightweight Windows vocabulary manager with a nostalgic classic-Windows interface.

## Features

- Vocabulary cards with translations, examples, explanations, grammar notes, synonyms, dialogues, levels, statuses, topics, and sections
- Search, filters, favorites, and flashcard practice
- Excel, CSV, and TSV import with column matching
- Excel export
- Optional image attachments for screenshots and visual associations
- Export individual cards as PNG images
- Incremental card loading and a quick Back to top button
- Local-first storage: vocabulary and attachments stay on the user's computer

## Installation

1. Download [`English-Vocab-DB-Setup.exe`](./English-Vocab-DB-Setup.exe).
2. Run the installer.

The installer is currently unsigned, so Windows SmartScreen may display a warning.

## Requirements

- Windows 10 or Windows 11, 64-bit
- Microsoft Edge

## Privacy

The application does not include or upload personal vocabulary cards. Card data and image attachments are stored locally in the application's browser profile on the user's computer.

## Updating

Close the application and run the newer installer over the existing version. Local vocabulary data is preserved.
